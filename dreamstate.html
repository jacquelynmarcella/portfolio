Dreamstate

React.js, MongoDB, Mogoose, Recharts, Node.js, Express.js, Material UI

Go to Site View on Github

Team Members: Antonia Villa and Ashton Christie

This MERN stack app provides data visualizations to help users understand their dreams. To analyze dreams, we used the AWS Sentiment Analysis API and data scraping. Then, we used Recharts to create data visualizations of individual dreams, sentiment over time, and commonly occuring symbols in the user's dreams. My primary contributions to this project were concepting the overall design direction, front end and responsive development, data scraping, and implenting the CRUD routes through Mongoose and React. Our group practiced pair programming in the initial sprint and managed workloads through Trello on subsequent sprints.
