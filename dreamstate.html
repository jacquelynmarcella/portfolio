Dreamstate

React.js, Node.js, Express.js, MongoDB, Mogoose, Recharts, Material UI

Go to Site View on Github

Team Members: Antonia Villa, Ashton Christie

This MERN stack app allows users to see the meaning of their dreams and track their dreams over time through data visualizations. Using the AWS Sentiment Analysis API and data scraping, users can see the overall sentiment of their dreams and what key symbols may mean. My primary contribtuion to this project was concepting the overall design direction, front end styling and responsive development, data scraping, and implenting the CRUD routes through the Mongoose ORM.

Our group practiced pair programming in the initial sprint and managed workloads through Trello on the following sprints.
