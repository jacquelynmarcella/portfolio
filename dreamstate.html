Dreamstate

React.js, MongoDB, Mongoose, Node.js, Express.js, Material UI

Live Site Github

Have you ever wondered what your dreams mean, or how they change over time? Dreamstate is an interactive MERN application that enables users to understand their dreams through data visualizations and keyword analysis.

Team Members: Antonia Villa and Ashton Christie

Primary Role: Front End Developer

Our objective was to provide engaging dream analysis with as minimal input as possible from the user. Working with the AWS Sentiment API and data scraping, we created a framework to analyze a userâ€™s dream based off sentiment and keywords. To help the user better understand the analysis, we used Recharts to create data visuals of individual dreams, sentiment over time, and commonly occuring symbols in the user's dreams. My primary contributions to this project were concepting the overall design direction, front end and responsive development, data scraping, and implementing CRUD routes through React and Mongo. Our group practiced pair programming in the initial sprint and managed workloads through Trello on subsequent sprints.
