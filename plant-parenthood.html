Plant Parenthood

Node.js, Express.js, PostgreSQL/Sequelize, jQuery, Semantic UI

Live Site Github

Role: Front and Back End Developer

If you weren't blessed with a green thumb, it might be time for a trip to Plant Parenthood. This app has a SQL database of over 200 houseplants with easy to understand care information. RESTful routing allows users to easily save plants, make public comments on plants, journal about their plants, as well as edit and delete these entries accordingly. I used web scraping to source images and care content from research studies. To create a better user experience, I translated the care data into simple icons. I used Semantic UI and jQuery to implement features such as the ability for a user to search the database and have results suggested in real time.
