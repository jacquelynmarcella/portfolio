Plant Parenthood

Node.js, Express.js, PostgreSQL, Sequelize, jQuery, Semantic UI

Go to Site View on Github

This app helps the casual houseplant collector to become a better plant "parent." I used web scraping to compile care information and images for over 200 houseplants into a PostgreSQL database. To create a better user experience, I used Semantic UI and jQuery to implement features such as the ability for a user to search the database and have results suggested in real time. This Node/Express app has RESTful routing, which allows users to easily save plants, make public comments on plants, journal about their plants, as well as edit and delete these entries accordingly.
