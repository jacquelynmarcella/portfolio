Plant Parenthood

Node.js, Express.js, PostgreSQL, Sequelize, jQuery, Semantic UI

Role: Front and Back End Developer
Live Site Github

If you weren't blessed with a green thumb, it might be time for a trip to Plant Parenthood. This Node/Express application has a SQL database of over 200 houseplants with easy to understand care information.

There are several resources for plant care, but most are filled with lengthy articles aimed at the experienced gardener. With the casual indoor plant collector in mind, I used web scraping to source images and care data from research studies. I translated the data into a simple icon system which allows the user to quickly interpret information about their plants. To make it easy for the user to find plants, I implemented a search feature with autosuggest through Semantic UI and jQuery. RESTful routing allows users to save plants, make public comments on plants, journal about their plants, as well as edit and delete these entries accordingly.
