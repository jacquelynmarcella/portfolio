Blackjack

jQuery, jQuery UI, Materialize

Go to Site View on Github

A JavaScript take on the classic card game. A user can place bets, split their deck, and even adjust aces when they go over 21. This game was developed mobile-first using Materialize to ensure that users can play on phones and desktop alike. Custom animations were added to help make the card game feel more real, such as flipping cards and moving when dealt.
