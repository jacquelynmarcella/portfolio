Blackjack

jQuery, jQuery UI, Materialize

Go to Site View on Github

A JavaScript take on the classic card game. A user can place bets, split their deck, and even adjust aces when they go over 21. Working with the Materialize framework, I developed this game with a mobile-first approach to ensure that users can play on phones and desktop alike. I added custom animations to the cards to help make the card game feel more engaging.
